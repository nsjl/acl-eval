About the ACLBot System

ACLBot is a conversational agent designed to help users explore scientific papers in the ACL Anthology. It is built on top of a large language model (OpenAI GPT-4) and has access to a structured knowledge graph (Neo4j) of ACL metadata and paper content.

The knowledge graph contains nodes representing papers, authors, events, volumes, passages, tasks, datasets, metrics and more. Relationships capture authorship, publication venues, paper content structure, and reported experimental results.

When a user asks a question, ACLBot may respond in one of three ways:

  • Using only the language model’s prior knowledge
  • Querying the knowledge graph with Cypher (Neo4j’s query language)
  • A hybrid approach combining LLM-generated responses and graph-based lookups

The goal of this evaluation is to study how well the system performs across different interaction types, and how useful graph-augmented responses are in practice.
