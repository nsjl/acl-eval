ACLBot Chat Evaluation

Welcome evaluators! This system, ACLBot, is a research chatbot that helps users explore and understand scientific papers from the ACL Anthology.

You do not need to read about the system internals to evaluate the chats. Just look at the conversation and judge how helpful the bot is.

If you're curious, you can read more about how the system works here.

ACLBot is powered by a large language model and connected to a Neo4j knowledge graph of papers, authors, venues, and topics. This lets it answer not only text-based questions but also perform graph-based reasoning â€” for example, finding citation links, research trends, and clusters of related work.

The goal of this evaluation is to assess how well the bot answers queries using both the language model and the knowledge graph. Each linked chat is a real interaction. Please read and evaluate using the provided form.
