ACLBot Chat Evaluation

Welcome evaluators! This system, ACLBot, is a research chatbot that helps users explore and understand scientific papers from the ACL Anthology.

If you're curious, you can read more about how the system works here.

ACLBot is powered by a large language model and connected to a Neo4j knowledge graph of papers, authors, venues, and topics. This lets it answer not only text-based questions but also query a knowledge graph â€” for example, finding specific authors, research trends, metrics etc. extracted from the ACL Anthology research corpus.

The goal of this evaluation is to assess how well the bot answers queries using both the language model and the knowledge graph.
